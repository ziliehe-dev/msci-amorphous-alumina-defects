# msci-amorphous-alumina-defects

Core scripts for the MSci dissertation:

**Defects in Disorder: Defect Evolution and Post-Relaxation Identifiability of Oxygen Frenkel Pairs in Amorphous Al2O3**

## Files
- `Analysis Before FPO Insertion.ipynb` — analysis of the defect-free reference structure before oxygen Frenkel-pair insertion
- `Analysis Immediately After FPO Insertion.ipynb` — analysis of the initial local structure immediately after oxygen Frenkel-pair insertion
- `Analysis of FPO SR.ipynb` — analysis of the structural relaxation of the oxygen Frenkel pair
- `FPO Insertion.ipynb` — construction of oxygen Frenkel-pair initial configurations
- `Melt.lmp` — LAMMPS input script for melt preparation
- `Quench.lmp` — LAMMPS input script for melt–quench preparation of amorphous Al2O3
- `Relaxation of FPO.in` — LAMMPS input script for post-insertion relaxation

## Notes
These scripts were used for defect insertion, trajectory analysis, and figure generation in the dissertation. Large molecular-dynamics trajectory files are not included in this repository.
